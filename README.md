# 律师办案智能助手 (Lawyer Assistant)

> 专为律师设计的办案全流程智能助手，整合案件管理、证据分析、法律文书、法律检索、庭审策略、内容创作、判决预测等专业能力。

## 项目概述

本项目整合自以下两个项目：
- **bananzhushou**：司法案例分析与裁判预测助手
- **wanneng**：律师办公全能助手

通过整合与优化，构建一个功能完整、高效专业的律师办案智能助手。

---

### 1. 案件管理
- 案件信息结构化提取
- 法定时限计算与提醒
- 材料清单生成
- 电子台账管理

### 2. 证据分析
- 证据收集清单生成
- 三性审查（真实性、合法性、关联性）
- 证据目录生成
- 质证意见起草

### 3. 法律文书
- 一审：起诉状、答辩状、代理词
- 二审：上诉状、二审答辩状、二审代理词
- 再审：再审申请书、再审答辩状
- 执行：执行申请、执行异议、恢复执行
- 保全：财产保全、证据保全、解除保全
- 调查取证：律师调查令申请书
- 其他：管辖权异议、追加第三人、鉴定申请、证人出庭
- 案由自动识别与针对性写作

### 4. 法律检索
- 法条检索
- 类案检索
- 案例分析（融合判决预测能力）
- 裁判规则总结

### 5. 庭审策略
- 庭前准备材料包
- 庭审话术
- 发问提纲
- 应对策略

### 6. 内容创作
- 小红书图文
- 抖音/视频号口播
- 多平台适配

### 7. 判决预测（核心新增）
- 案例学习与规则萃取
- 新案匹配与预测分析
- 风险提示与免责声明

## 整合优化说明

### 精简与优化
1. **Agent → Skill**：将 wanneng 项目的 6 个 Agent 统一为 Skill 格式
2. **能力融合**：将 bananzhushou 的案例分析与判决预测能力融入 legal-researcher
3. **独立预测**：判决预测作为独立 Skill，保留完整的三阶段处理流程

### 功能去重
1. **案例分析统一入口**：所有案例相关功能通过 `legal-researcher` 处理
2. **裁判规则内嵌**：规则萃取能力整合到法律检索功能中
3. **预测能力独立**：判决预测独立 Skill，但可调用检索结果
