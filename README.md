# 律师办案智能助手 (Lawyer Assistant)

> 专为律师设计的办案全流程智能助手，整合案件管理、证据分析、法律文书、法律检索、庭审策略、内容创作、判决预测等专业能力。

## 项目概述

本项目整合自以下两个项目：
- **bananzhushou**：司法案例分析与裁判预测助手
- **wanneng**：律师办公全能助手

通过整合与优化，构建一个功能完整、高效专业的律师办案智能助手。

## 整合优化说明

### 精简与优化
1. **Agent → Skill**：将 wanneng 项目的 6 个 Agent 统一为 Skill 格式
2. **能力融合**：将 bananzhushou 的案例分析与判决预测能力融入 legal-researcher
3. **独立预测**：判决预测作为独立 Skill，保留完整的三阶段处理流程

### 功能去重
1. **案例分析统一入口**：所有案例相关功能通过 `legal-researcher` 处理
2. **裁判规则内嵌**：规则萃取能力整合到法律检索功能中
3. **预测能力独立**：判决预测独立 Skill，但可调用检索结果
