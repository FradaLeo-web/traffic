# 律师办案智能助手 (Lawyer Assistant)

> 专为律师设计的办案全流程智能助手，整合案件管理、证据分析、法律文书、法律检索、庭审策略、内容创作、判决预测等专业能力，覆盖律师90%以上日常工作场景。

## 项目概述

本项目整合自以下两个项目：
- **bananzhushou**：司法案例分析与裁判预测助手
- **wanneng**：律师办公全能助手

通过持续整合与优化，构建了一个功能完整、高效专业、适配全国司法实践的律师办案智能助手。

### 🎯 地区适配规则
**默认优先适配山西省标准，山西省内优先适配太原市标准，同时支持全国31个省市自治区规则切换：**
- ✅ 内置全国各省市赔偿标准、量刑指导意见、地方司法政策
- ✅ 用户可选择所在地区，自动适配当地最新裁判尺度、赔偿计算标准
- ✅ 赔偿计算类：支持各省最新统计数据自动调用（残疾赔偿金、误工费、护理费等）
- ✅ 刑事案件：支持各省量刑指导意见实施细则自动适配
- ✅ 民事案件：支持各地高院司法文件、会议纪要、裁判倾向参考

详细规则见：[地区适配规则.md](./地区适配规则.md)

## 使用方式

### 🚀 快速开始
1. 打开 Claude Code / OpenClaw 并进入 `lawyer-assistant` 项目目录
2. 直接输入 `/skill 技能名` 即可唤起对应功能
3. 按提示输入案件相关信息，自动生成专业输出

**示例：**
```
/skill 伤残等级评估
请描述伤者伤情：左踝关节骨折，左足距骨骨折，左足舟骨骨折...
```
3秒即可生成专业伤残等级评估报告、鉴定指导和赔偿参考。

#### 1. 判决预测（升级）

- 核心影响因素权重分析，提示影响判决的关键要点
