## 🎯 使用指南

### 1. 法律法规查询

法律知识按领域分类存储，每个法律文件为一个Markdown文件。

### 2. 案例检索

典型案例按案由分类，每个案例文件包含：
案情、争议焦点、裁判结果、裁判要点、法律适用

### 3. 文书模板

文书模板包含可编辑的占位符，便于快速生成。

### 4. 计算标准

赔偿标准包含计算公式、适用标准、计算示例、数据来源。

### 5. 办案流程

流程文档包含办案阶段划分、各阶段时限要求、所需材料清单、注意事项。

## 📝 文件命名规范

1. **法律法规**：`法律名称.md`
2. **案例**：`年份-案号-案件类型.md`
3. **文书模板**：`文书类型-具体名称.md`
4. **标准**：`标准类型-年份-地区.md`
5. **流程**：`流程类型-具体名称.md`

## 🔍 检索建议

1. **按领域检索**：根据案件类型进入对应目录
2. **关键词搜索**：使用文件内容关键词进行全文搜索
3. **索引导航**：查阅 `index.md` 获取完整索引
