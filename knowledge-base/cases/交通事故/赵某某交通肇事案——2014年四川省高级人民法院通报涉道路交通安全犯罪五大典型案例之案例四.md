# 赵某某交通肇事案——2014年四川省高级人民法院通报涉道路交通安全犯罪五大典型案例之案例四

1/1赵某某交通肇事案——2014年四川省高级人民法院通报涉道路交通安
全犯罪五大典型案例之案例四
案 由 ：刑事/危害公共安全罪/交通肇事罪
【基本案情】
被告人赵某某，男，汉族，1972年3月出生，小学文化，四川省泸州市人。2011年2月因犯交通肇
事罪被判处有期徒刑十个月，缓刑一年。
2013年10月13日18时35分，被告人赵某某无证驾驶中型自卸货车（核载1.998T，实载15.06T）
从叙永往纳溪区方向行驶，行至泸州市护国镇G321线1770km+50m处，与相向行驶的由易某某驾驶的小型
轿车发生擦挂，后又与相向行驶的由伍某某驾驶的大中型拖拉机尾部相撞，致拖拉机失控冲向右侧人行
道，致行人卢某某、陈某某当场死亡，拖拉机驾驶员伍某某、乘车人杨某某、伍某某、行人喻某某、夏某
某受伤，三车受损的交通事故。经认定，赵某某承担此交通事故全部责任。后被告人赵某某到交警队投案
自首。
【裁判结果】
一审法院审理认为，被告人赵某某违反交通运输管理法规，无证驾驶严重超载的机动车在道路上行
驶，造成2人死亡,5人受伤的交通事故，其行为已构成交通肇事罪，依法应负刑事责任。赵某某交通肇事
致2人死亡，并承担事故全部责任，依法应处三年以上七年以下有期徒刑。赵某某有自首情节，可以从轻
或减轻处罚；赵某某有犯罪前科，应酌情从重处罚。据此认定被告人赵某某犯交通肇事罪，判处有期徒刑
三年十个月。本案在法定期限内没有上诉、抗诉，现已生效。
