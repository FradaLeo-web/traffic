# 倪某某、陈某某、冯某某、王某某、张某某以危险方法危害公共安全案——2014年四川省高级人民法院通报涉道路交通安全犯罪五大典型案例之案例五

1/2倪某某、陈某某、冯某某、王某某、张某某以危险方法危害公共安全
案——2014年四川省高级人民法院通报涉道路交通安全犯罪五大典型
案例之案例五
案 由 ：刑事/危害公共安全罪/以危险方法危害公共安全罪
【基本案情】
被告人倪某某，男，汉族，1986年6月出生，小学文化，四川省成都市人。
被告人陈某某，男，汉族，1970年4月出生，小学文化，四川省成都市人。（概述前科情况）
被告人冯某某，男，汉族，1988年11月出生，小学文化，四川省成都市人。
被告人王某某，男，汉族，1990年8月出生，小学文化，四川省成都市人。
被告人张某某，男，汉族，1992年1月出生，小学文化，四川省成都市人。
2012年12月至2013年1月，为牟取交通事故赔偿款和保险赔偿金，被告人倪某某、陈某某、冯某
某、王某某、张某某共谋后，先后驾驶倪某某经营的汽车修理店内待维修的奥迪Q7越野车、别克凯越轿
车以及冯某某的宝来出租车及借来的雪铁龙轿车、捷达轿车，在成都市龙泉驿区、新都区、金堂县及简阳
市平泉镇、贾家镇等地公路上，针对不特定的汽车驾驶员和摩托车司机，以故意开车撞击或紧急制动的方
式，制造车辆受损或人员受伤的交通事故，骗取车辆保险金，并胁迫被害人赔偿车辆损失及医疗费用，共
计获利3万余元。
【裁判结果】
一审法院审理认为，被告人倪某某、陈某某、冯某某、王某某、张某某结伙在公共道路上，针对正在
行驶中的不特定的机动车辆，采用故意撞击或紧急制动的危险方法制造交通事故，以此牟取非法利益的行
为，已触犯刑律，构成了以危险方法危害公共安全罪，且系共同犯罪，依法应当追究刑事责任。倪某某、
陈某某、冯某某在共同犯罪中起主要作用，系主犯，依法应当按照其所参与或者组织、指挥的全部犯罪处
罚；王某某、张某某在共同犯罪中起次要作用，系从犯，依法应当从轻、减轻或者免除处罚。冯某某到案
后，协助公安机关抓获其他被告人，有立功表现，可从轻或减轻处罚。倪某某、陈某某、冯某某、王某
某、张某某到案后均如实供述罪行且当庭自愿认罪，依法可以从轻处罚。倪某某、张某某积极赔偿部分被
害人的经济损失并取得谅解，可酌情从轻处罚。陈某某有犯罪前科，可酌情从重处罚。据此认定倪某某犯
以危险方法危害公共安全罪，判处有期徒刑四年；被告人陈某某犯以危险方法危害公共安全罪，判处有期
徒刑四年；被告人冯某某犯以危险方法危害公共安全罪，判处有期徒刑三年六个月；被告人王某某犯以危
险方法危害公共安全罪，判处有期徒刑一年六个月；被告人张某某犯以危险方法危害公共安全罪，判处有
期徒刑一年，缓刑一年六个月。本案在法定期限内没有上诉、抗诉，现已生效。
