### 输入
```

案由：劳动争议
诉讼请求：
1. 请求确认原告与被告存在劳动关系（2019年3月至2024年8月）
2. 请求被告支付拖欠工资 85,000元
3. 请求被告支付违法解除劳动合同赔偿金 52,000元（月平均工资6500元×4个月×2）
```

## 测试3：材料话术生成

### 输入
```
案件类型：劳动争议（工伤赔偿）
当前阶段：准备立案
已收集材料：
1. 工伤认定决定书
2. 劳动能力鉴定结论书（玖级伤残）
3. 医疗费发票（共计32,500元）
4. 病历资料

还需要补充：
- 劳动关系证明（劳动合同/入职材料）
- 工资收入证明（事故前12个月）
- 护理费证明
- 误工证明
```

### 期望输出
```
【向客户索要材料话术】

张先生/女士，您好！

根据您提供的案件材料，我们已整理好现有证据，但为确保案件顺利立案并获得合理赔偿，还需补充以下材料：

1. **劳动关系证明**
   - 材料：劳动合同、入职登记表、工作证、考勤记录等
   - 原因：证明您与用人单位存在劳动关系
   - 提交方式：电子版扫描件或照片

2. **工资收入证明**
   - 材料：事故前12个月工资银行流水、工资条、个税证明
   - 原因：用于计算误工费和工伤赔偿标准
   - 提交方式：银行流水加盖公章，其他提供原件核对

3. **护理费证明**
   - 材料：护理人员身份证明、误工证明、收入证明（如有）
   - 原因：主张住院期间护理费
   - 提交方式：提供原件核对

4. **误工证明**
   - 材料：单位出具的误工证明、考勤记录
   - 原因：主张停工留薪期工资
   - 提交方式：加盖单位公章的原件

请您尽快准备以上材料，感谢配合！
```
