# 证据分析专家 - 测试用例

## 测试1：证据三性审查

### 输入
```
案由：民间借贷纠纷
诉讼请求：请求被告偿还借款本金 50,000元及利息（按年利率15%计算，从2023年1月1日起至实际清偿之日止）

已收集证据：
1. 借条（内容：今借到张三人民币50,000元，约定年利率15%，借款人李四，2022年12月1日）- 原件
2. 银行转账记录（2022年12月1日，张三账户转入李四账户50,000元）- 盖有银行公章
3. 微信聊天记录（2023年3月至8月，张三向李四催款，李四表示无力偿还）- 手机截屏
4. 录音（2023年9月张三与李四的通话录音，李四承认欠款但表示无力偿还）- 手机录音
5. 证人证言（邻居王五证明曾听李四说过向张三借钱）- 书面证言，证人未出庭
```

### 期望输出
```
【证据三性审查表】

| 证据 | 真实性 | 合法性 | 关联性 | 瑕疵说明 | 补正建议 |
|------|--------|--------|--------|----------|----------|
| 借条 | ✓ 真实 | ✓ 合法 | ✓ 关联 | - | - |
| 银行转账记录 | ✓ 真实 | ✓ 合法 | ✓ 关联 | - | - |
| 微信聊天记录 | ⚠ 待核实 | ✓ 合法 | ✓ 关联 | 需提供原始载体核对 | 提交手机原件核对 |
| 录音 | ⚠ 待核实 | ✓ 合法 | ✓ 关联 | 需证明录音未经剪辑 | 提供原始录音文件 |
| 证人证言 | ✗ 不予采信 | ✓ 合法 | ✓ 关联 | 证人未出庭，无法质证 | 申请证人出庭 |

【审查说明】
- 借条+转账记录：形成完整证据链，证明借贷关系成立
- 微信聊天记录：需核对原始载体，确认真实性
- 录音：需提供原始录音文件，必要时申请声纹鉴定
- 证人证言：因证人未出庭，证明力较低，建议申请出庭

【证据目录】
证据1 借条 原告 证明借贷关系成立 第1页
证据2 银行转账记录 原告 证明款项已实际交付 第2页
证据3 微信聊天记录 原告 证明催款事实 第3-5页
证据4 录音 原告 证明被告承认欠款 第6页
```

## 测试2：质证意见起草

### 输入
```
案由：机动车交通事故责任纠纷
我方：原告（受害人）
对方：被告（肇事司机）

对方可能提交的证据：
1. 车速鉴定报告（证明当时车速为65km/h，未超速）
2. 事故现场照片（证明原告闯红灯）
3. 车辆保险单（证明已投保交强险和商业三者险100万元）
4. 医疗费发票复印件（证明已垫付3万元医疗费）
```

### 期望输出
```
【针对对方证据的质证意见】

对方证据1：车速鉴定报告
一、对真实性的质证
对真实性有异议，需核实鉴定机构资质、鉴定程序是否合法
二、对合法性的质证
对合法性有异议，鉴定时机是否恰当、鉴定材料是否完整
三、对关联性的质证
对关联性有异议，车速与事故发生无直接因果关系
四、对证明力的质证
该证据不能证明被告无过错，不能减轻其责任

---

对方证据2：事故现场照片
一、对真实性的质证
对真实性有异议，无法证明照片拍摄时间和地点
二、对合法性的质证
对合法性有异议，照片来源不明
三、对关联性的质证
对关联性有异议，照片不能证明原告闯红灯
四、对证明力的质证
该证据为单方拍摄，证明力较低

---

对方证据3：车辆保险单
一、对真实性的质证
无异议
二、对合法性的质证
无异议
三、对关联性的质证
对关联性有异议，保险情况与责任认定无关
四、对证明力的质证
对证明力有异议，保险不能免除侵权责任

---

对方证据4：医疗费发票复印件
一、对真实性的质证
对真实性有异议，需核对原件
二、对合法性的质证
无异议
三、对关联性的质证
无异议
四、对证明力的质证
需核对原件后确认
```
