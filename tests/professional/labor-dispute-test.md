# 劳动争议 - 测试用例

## 测试1：工伤赔偿计算（山西省标准）

### 输入
```
案件情况：
- 劳动者：张某，男，42岁
- 用人单位：山西某煤矿公司
- 受伤时间：2024年5月
- 工伤认定：已认定为工伤
- 伤残等级：七级伤残
- 受伤前12个月平均工资：8000元/月
- 住院时间：30天
- 停工留薪期：6个月
- 劳动关系解除时间：2024年12月（伤后7个月）
- 统筹地区上年度职工月平均工资：6500元
- 供养亲属：无（未婚，无子女，父母已故）
```

### 期望输出
```
【工伤赔偿计算清单】

一、一次性伤残补助金
- 计算：本人工资 × 13个月
- 结果：8000 × 13 = 104,000元

二、一次性工伤医疗补助金（山西省）
- 计算：统筹地区上年度职工月平均工资 × 24个月
- 结果：6500 × 24 = 156,000元

三、一次性伤残就业补助金（山西省）
- 计算：本人工资 × 15个月
- 结果：8000 × 15 = 120,000元

四、停工留薪期工资
- 计算：原工资福利待遇 × 6个月
- 结果：8000 × 6 = 48,000元

五、护理费
- 计算：当地护工标准 × 住院天数
- 假设：150元/天 × 30天 = 4,500元

六、住院伙食补助费
- 计算：当地标准 × 住院天数
- 假设：100元/天 × 30天 = 3,000元

七、医疗费（凭票据实结算）
- 金额：待结算

八、交通费（凭票据实结算）
- 金额：待结算

【赔偿合计】（不含医疗费和交通费）
104,000 + 156,000 + 120,000 + 48,000 + 4,500 + 3,000 = 435,500元
```

## 测试2：经济补偿金计算

### 输入
```
案件情况：
- 劳动者：李某
- 用人单位：某科技公司
- 工作年限：2019年3月 至 2024年8月（共5年5个月）
- 解除前12个月平均工资：12000元/月
- 解除原因：用人单位提出，协商一致解除
- 劳动者是否提前通知：已提前30天通知
```

### 期望输出
```
【经济补偿金计算】

一、计算年限
- 工作年限：5年5个月
- 折算：按5.5个月计算

二、计算基数
- 月平均工资：12000元
- 是否高于当地平均工资3倍：否（假设当地平均工资为8000元）
- 计算基数：12000元

三、计算公式
经济补偿金 = 月工资 × 工作年限
           = 12000 × 5.5
           = 66,000元

【结果】
经济补偿金：66,000元（N=5.5）

【说明】
- 工作年限不满6个月按0.5个月计算
- 如用人单位违法解除，赔偿金为2倍，即132,000元（2N）
```

## 测试3：劳动关系认定

### 输入
```
案件情况：
赵某主张与A公司存在劳动关系，A公司予以否认。

赵某提供的证据：
1. 微信工作群聊天记录（显示A公司主管安排工作）
2. 工资转账记录（每月固定日期转账，备注"工资"）
3. 考勤打卡记录（公司门禁系统记录）
4. 工作服（印有A公司logo）
5. 介绍人证明（证明赵某经介绍到A公司工作）

A公司主张：
赵某系B劳务派遣公司派遣到A公司工作的，劳务派遣协议由A公司与B公司签订。
```

### 期望输出
```
【劳动关系认定分析】

一、审查要点
根据劳动和社会保障部〔2005〕12号文，同时具备以下情形的，劳动关系成立：
1. 用人单位和劳动者符合法律、法规规定的主体资格
2. 用人单位依法制定的各项劳动规章制度适用于劳动者
3. 劳动者受用人单位的劳动管理
4. 劳动者提供的劳动是用人单位业务的组成部分

二、证据分析

| 证据 | 证明内容 | 证明力 |
|------|----------|--------|
| 微信工作群 | 接受A公司管理 | 中 |
| 工资转账记录 | 定期领取报酬 | 强 |
| 考勤打卡记录 | 接受A公司管理 | 强 |
| 工作服 | 为A公司提供劳动 | 中 |
| 介绍人证言 | 证明入职情况 | 弱 |

三、劳务派遣抗辩审查
- A公司需提供与B公司的劳务派遣协议
- 如无法提供，应承担举证不能的后果
- 即便存在劳务派遣，如用工单位未依法告知劳动者，仍可能认定事实劳动关系

四、结论
赵某与A公司存在事实劳动关系的可能性较大。
建议代理方向劳动仲裁委员会申请调查取证，要求A公司提供劳务派遣协议。
```
