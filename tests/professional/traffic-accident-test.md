# 交通事故 - 测试用例

## 测试1：赔偿计算与保险理赔

### 输入
```
案件情况：
- 事故时间：2024年6月15日
- 责任划分：司机刘某（主责70%）、行人孙某（次责30%）
- 刘某车辆：交强险 + 商业三者险100万元（不计免赔）
- 孙某伤情：
  - 住院治疗25天
  - 医疗费：68,000元
  - 误工费：120元/天 × 90天 = 10,800元
  - 护理费：150元/天 × 45天 = 6,750元
  - 营养费：30元/天 × 45天 = 1,350元
  - 交通费：800元
  - 残疾赔偿金：城镇居民人均可支配收入 41,727元 × 20年 × 10% = 83,454元（十级伤残）
  - 被扶养人生活费：父亲（65岁，城镇）- 30,143元 × 15年 ÷ 2 × 10% = 22,607元
  - 精神损害抚慰金：5,000元

- 刘某车辆损失：12,000元（已定损）
- 孙某财产损失：手机损坏 3,000元
```

### 期望输出
```
【赔偿计算清单】

一、孙某损失（刘某应承担部分）

1. 医疗费：68,000元 × 70% = 47,600元
2. 误工费：10,800元 × 70% = 7,560元
3. 护理费：6,750元 × 70% = 4,725元
4. 营养费：1,350元 × 70% = 945元
5. 交通费：800元 × 70% = 560元
6. 残疾赔偿金：83,454元 × 70% = 58,418元
7. 被扶养人生活费：22,607元 × 70% = 15,825元
8. 精神损害抚慰金：5,000元（酌定）

【交强险限额内的赔偿】
- 死亡伤残赔偿限额：180,000元
  - 误工费 7,560元
  - 护理费 4,725元
  - 交通费 560元
  - 残疾赔偿金 58,418元
  - 被扶养人生活费 15,825元
  - 精神损害抚慰金 5,000元
  - 合计：92,088元（未超限额）

- 医疗费用赔偿限额：18,000元
  - 医疗费 47,600元
  - 营养费 945元
  - 合计：48,545元（超限）
  
- 财产损失赔偿限额：2,000元
  - 手机损失 3,000元 × 70% = 2,100元（超限）

【交强险赔偿后余额】
- 医疗费用：(47,600+945) - 18,000 = 30,545元
- 财产损失：2,100 - 2,000 = 100元
- 合计：30,645元

【商业三者险赔偿】
- 交强险赔付后余额 × 责任比例 = 30,645元 × 100% = 30,645元

【保险公司总计赔偿】
- 交强险：92,088 + 18,000 + 2,000 = 112,088元
- 商业三者险：30,645元
- 合计：142,733元

---

二、刘某车辆损失（孙某应承担部分）
- 车辆损失：12,000元 × 30% = 3,600元
- 由刘某向孙某追偿
```

## 测试2：伤残等级预估

### 输入
```
伤情描述：
- 伤者因交通事故导致左胫骨平台骨折
- 住院行左胫骨平台骨折内固定术
- 膝关节活动度：伸直0度，屈曲90度（正常伸直0度，屈曲135度）
- 膝关节功能丧失约33%
- 伤后6个月复查
```

### 期望输出
```
【伤残等级预估】

一、损伤情况分析
- 损伤部位：左胫骨平台骨折
- 治疗方式：内固定手术
- 功能障碍：膝关节活动度受限

二、对照标准
《人体损伤致残程度分级》十级级条款：
- 5.10.6.11：四肢任一大关节功能丧失25%以上

三、膝关节功能计算
- 正常屈伸范围：0°-135°
- 实际屈伸范围：0°-90°
- 功能丧失程度：(135-90)/135 = 33.3%

四、预估结论
- 伤残等级：十级伤残
- 依据：膝关节功能丧失约33%，符合十级标准
- 说明：内固定物取出后功能可能进一步恢复，建议取内固定后复查再鉴定

五、误工期、护理期、营养期参考
- 误工期：120-180天
- 护理期：60-90天
- 营养期：30-60天

六、注意事项
1. 鉴定时机：一般在治疗终结后3-6个月
2. 内固定取出：取内固定后可再次评估伤残等级
3. 建议申请法院委托鉴定，避免单方鉴定争议
```
