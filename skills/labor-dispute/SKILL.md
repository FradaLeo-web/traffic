# 劳动争议律师技能

专注于劳动合同、工资报酬、工伤赔偿、社会保险等劳动争议纠纷，提供从证据收集到诉讼执行的全流程法律服务。

## 核心能力

- 劳动关系认定（事实劳动关系、劳务关系、平台用工）
- 工资报酬争议（加班费、年终奖、经济补偿金、赔偿金）
- 工伤赔偿（工伤认定、劳动能力鉴定、赔偿计算）
- 社会保险纠纷（社保缴纳、待遇支付）
- 劳动合同解除/终止（违法解除、经济补偿、赔偿金）

## 输出格式规范

### 1. 劳动关系分析

**争议焦点**：是否存在劳动关系、劳动关系起止时间

**认定标准**：
- 用人单位和劳动者符合法律、法规规定的主体资格
- 用人单位依法制定的各项劳动规章制度适用于劳动者
- 劳动者受用人单位的劳动管理
- 劳动者提供的劳动是用人单位业务的组成部分

**法律依据**：
- 《劳动合同法》第2条、第7条
- 劳动部〔2005〕12号文

### 2. 工资报酬计算

**加班费计算**：
- 加班费基数：正常出勤工资（剔除加班费、津贴、奖金）
- 工作日延时：150%
- 休息日加班：200%
- 法定节假日：300%

**经济补偿金/赔偿金**：
- 经济补偿金：N（工作年限×月工资）
- 赔偿金：2N（违法解除）
- 月工资标准：解除前12个月平均工资（含奖金、津贴）

### 3. 工伤赔偿清单

**工伤认定条件**：
- 工作时间和工作场所内，因工作原因受到事故伤害
- 工作时间前后在工作场所内，从事与工作有关的预备性/收尾性工作受到事故伤害
- 因工外出期间，由于工作原因受到伤害或发生事故下落不明
- 在上下班途中，受到非本人主要责任的交通事故

**赔偿项目**：
- **一次性伤残补助金**：本人工资×7-27个月（按伤残等级）
- **一次性工伤医疗补助金**：统筹地区标准×月数
- **一次性伤残就业补助金**：统筹地区标准×月数
- **停工留薪期工资**：原工资福利待遇不变
- **停工留薪期护理费**：按统筹地区标准

**工伤认定申请**：

**申请时限**：
- 用人单位申请：事故发生之日起30日内
- 个人申请：事故发生之日起1年内（用人单位未申请时）

**申请部门**：
- 市级人力资源和社会保障局（工伤认定科）

**申请材料**：
1. 工伤认定申请表
2. 劳动关系证明（劳动合同、工资条、社保记录等）
3. 医疗诊断证明
4. 事故证明材料（事故现场照片、证人证言、公安机关证明等）
5. 身份证复印件

**认定流程**：
1. 提交申请材料
2. 受理审查（15日内）
3. 调查核实（必要时）
4. 作出认定决定（60日内，可延长30日）
5. 送达认定决定书

**劳动能力鉴定申请**：

**申请时机**：
- 停工留薪期满
- 伤情相对稳定后

**申请部门**：
- 山西省劳动能力鉴定委员会（设区市劳动能力鉴定委员会）

**申请材料**：
1. 劳动能力鉴定申请表
2. 工伤认定决定书
3. 医疗病历资料（完整的住院病历、检查报告等）
4. 身份证复印件

**鉴定标准**：
- 《劳动能力鉴定 职工工伤与职业病致残等级》(GB/T16180-2014)

**鉴定流程**：
1. 提交申请材料
2. 受理审查
3. 医学检查（指定医院）
4. 专家评审
5. 作出鉴定结论（60日内）

**不服处理**：
- 收到鉴定结论15日内申请再次鉴定
- 向省级劳动能力鉴定委员会申请

**山西省特殊规定**：
- 鉴定费用由工伤保险基金支付
- 再次鉴定结论为最终结论
- 需要定期复查的伤情，应按规定复查

### 6. 仲裁前置程序

**时效规定**：
- 劳动仲裁：知道权利被侵害之日起1年
- 仲裁裁决：60日内（案情复杂可延长30日）

**仲裁流程**：
1. 申请仲裁（提交仲裁申请书、证据）
2. 受理与通知（5日内决定是否受理）
3. 开庭审理（举证期限、庭审辩论）
4. 调解优先（庭前、庭中调解）
5. 裁决作出（45日内，可延长15日）

**诉讼衔接**：
- 收到裁决书之日起15日内起诉
- 追索劳动报酬、工伤医疗费、经济补偿金可申请先予执行

### 7. 诉讼策略与风险提示

**管辖选择**：
- 劳动仲裁：用人单位所在地或劳动合同履行地
- 诉讼：用人单位所在地（劳动仲裁前置）

**鉴定时机**：
- 工伤认定：事故发生后30日内单位申请，1年内个人申请
- 劳动能力鉴定：停工留薪期满或伤情相对稳定

**风险点**：
- 超过仲裁时效（1年）
- 工资基数认定争议（是否包含补贴、奖金）
- 举证责任倒置（用人单位掌握考勤、工资记录）
- 违法解除的认定标准（程序合法性、事实依据）

## 工亡赔偿标准（山西省）

### 一次性工亡补助金
- **标准**：上一年度全国城镇居民人均可支配收入 × 20个月（全国统一）
- **2025年预估标准**：约 98万元（基于2024年全国数据：53921元×20）
- **法律依据**：《工伤保险条例》第39条

### 丧葬补助金
- **标准**：统筹地区上年度职工月平均工资 × 6个月
- **山西省2024年平均工资**：约7500元/月
- **预估金额**：约4.5万元
- **法律依据**：《工伤保险条例》第39条

### 供养亲属抚恤金
- **配偶**：因工死亡职工本人工资 × 40%/月
- **其他亲属**：每人 × 30%/月，总额不超过100%
- **孤寡老人或孤儿**：在上述标准基础上增加10%
- **领取年龄限制**：
  - 配偶：男满60周岁、女满55周岁
  - 子女：未满18周岁或已满18周岁因残疾/上学不能独立生活
  - 父母：男满60周岁、女满55周岁且无养老金
- **法律依据**：《工伤保险条例》第39条、《因工死亡职工供养亲属范围规定》

### 山西省特殊规定
- **供养亲属抚恤金调整**：每年按省统计局公布的职工平均工资调整
- **最低保障**：低于当地最低工资标准的，按最低工资标准发放

## 挂靠经营案件特殊处理策略

### 法律依据
**《最高人民法院关于审理工伤保险行政案件若干问题的规定》第三条第一款第五项**：
> "个人挂靠其他单位对外经营，其聘用的人员因工伤亡的，被挂靠单位为承担工伤保险责任的单位。"

**最高人民法院公报案例（2022年第11期）裁判要旨**：
> "在车辆挂靠关系中，被挂靠人向挂靠人收取挂靠费，应与挂靠人共同承担经营运输风险，仅以协议约定不能免除其作为被挂靠人应承担的风险和责任。个人挂靠其他单位对外经营，其聘用的人员因工伤亡，被挂靠单位以不存在劳动关系为由，主张不承担工伤保险责任的，人民法院不予支持。"

### 适用条件
- 车辆登记在被挂靠单位名下 ✓
- 车辆以被挂靠单位名义运营（有保险） ✓
- 个人（车主）聘用司机 ✓
- 司机因工死亡 ✓

### 举证责任分配
**公司（被挂靠单位）需要证明**：
- 车辆已卖给个人（买卖合同、付款凭证、过户协议）
- 个人车主的身份信息
- 个人车主与司机的雇佣关系
- 个人车主的经营控制（派单、工资发放等）
- 挂靠协议（如存在）
- 管理费收取记录

**如公司无法证明上述事项，承担举证不能的后果**：
- 认定车辆仍属于公司
- 认定司机受公司管理、由公司支付报酬
- 认定事实劳动关系成立

### 双成功路径策略
**路径一：认定劳动关系后认定工伤**
- 优势：符合常规流程，能同时解决劳动关系和工伤问题
- 风险：公司可能抗辩劳务关系，需充分证据证明从属性

**路径二：适用挂靠责任，不要求劳动关系**
- 优势：即使仲裁未认定劳动关系，工伤认定仍可能成立
- 适用依据：最高法院司法解释第3条第5项
- 参考：最高法院公报案例明确支持此路径

### 代理策略建议
1. **优先主张劳动关系**，提供充分的从属性证据
2. **同时准备挂靠责任路径**，作为保底方案
3. **仲裁阶段**：要求公司举证车辆买卖和挂靠关系
4. **工伤认定阶段**：如劳动关系认定困难，直接适用挂靠责任
5. **行政诉讼阶段**：重点论证车辆以公司名义运营，公司应承担责任

## 突发疾病死亡（48小时规则）

### 法律依据
**《工伤保险条例》第十五条第一项**：
> "职工在工作时间和工作岗位，突发疾病死亡或者在48小时之内经抢救无效死亡的，视同工伤。"

### 认定条件（四要件）
1. **工作时间**：正常工作时间、加班时间、值班时间
2. **工作岗位**：劳动者履行工作职责的场所（包括车辆驾驶位、外勤现场等）
3. **突发疾病**：工作期间突发的疾病，包括心血管疾病、脑血管疾病等
4. **48小时内死亡**：从医疗机构初次诊断时间起算，48小时内死亡

### "48小时"起算点
- **标准**：以医疗机构初次诊断时间为起算点
- **证据**：门（急）诊病历记录、120出车记录
- **认定**：从初次诊断时间到死亡时间不超过48小时

### 关键证据收集
**工作时间证明**：
- 考勤记录、派单记录、排班表
- 工作群聊天记录、派车单
- 事发现场照片、监控视频

**工作岗位证明**：
- 工作证、IC卡、派车单
- 车辆登记证明（司机在车辆上猝死）
- 工作地点证明

**死亡时间证明**：
- 居民死亡医学证明书
- 医院抢救记录
- 120出车记录和到达时间
- 尸检报告（如有）

**排除他杀证明**：
- 公安机关情况说明
- 法医鉴定报告
- 尸检报告

### 山西省特殊规定
- **工伤预防补偿费**：符合条件的企业可申请工伤预防补偿
- **死亡性质认定**：需排除非工作原因导致的死亡（如自杀、吸毒等）

## 劳动仲裁代理词模板（工亡案件）

### 基本结构

#### 1. 案件事实部分
- 入职及工作情况（入职时间、工作岗位、工作内容）
- 报酬发放情况（工资标准、发放方式、发放记录）
- 猝死情况（时间、地点、抢救经过、死亡原因）
- 排他杀证明（公安机关证明、法医鉴定）

#### 2. 代理意见

**论点一：确认存在事实劳动关系**
- 人格从属性：接受单位管理和监督、不能随意拒单、请假需批准
- 组织从属性：持有单位工作证/IC卡、加入单位工作群、执行单位业务
- 经济从属性：由单位支付劳动报酬、长期稳定结算、为主要收入来源
- 业务从属性：劳动内容是单位业务组成部分

**论点二：单位抗辩理由不成立**
- 车辆买卖主张缺乏证据（无买卖合同、无付款凭证、未过户）
- 挂靠关系无法成立（无挂靠协议、无管理费收取、无实际车主控制）
- 工资发放矛盾（如为车主雇佣，为何由单位财务人员发放）
- 举证责任：单位需证明车辆买卖和挂靠关系，承担举证不能后果

**论点三：退一步讲，单位也应承担工伤责任**
- 引用最高法院司法解释第3条第5项
- 车辆登记在单位名下，以单位名义运营
- 即使不存在劳动关系，单位也应承担工伤责任

**论点四：符合工亡认定条件**
- 工作时间：连续工作、在工作岗位上
- 工作岗位：猝死地点为工作场所（车辆驾驶位等）
- 工作原因：突发疾病，与工作存在因果关系
- 48小时内死亡：从初次诊断到死亡不超过48小时
- 排他杀：公安机关排除他杀

#### 3. 证据清单
- 劳动关系证据（工作证、IC卡、工作群记录、工资发放记录）
- 工亡事实证据（死亡证明、抢救记录、现场照片）
- 车辆登记证明（行驶证、保险单）
- 排他杀证明（公安机关证明）

#### 4. 法律依据
- 《劳动合同法》第2条、第7条
- 劳动部〔2005〕12号文（劳动关系认定标准）
- 《工伤保险条例》第15条第1项（突发疾病死亡视同工伤）
- 最高人民法院关于审理工伤保险行政案件规定第3条第5项

## 工亡赔偿计算工具说明

### 一次性工亡补助金计算
**计算公式**：
```
一次性工亡补助金 = 上一年度全国城镇居民人均可支配收入 × 20个月
```

**查询方式**：
- 国家统计局官网（http://www.stats.gov.cn/）
- 国务院公报（每年公布）
- 人社部工伤保险司官网

**示例计算**（基于2024年数据）：
- 2024年全国城镇居民人均可支配收入：53921元
- 一次性工亡补助金：53921 × 20 = 1,078,420元（约107.8万元）

### 丧葬补助金计算
**计算公式**：
```
丧葬补助金 = 统筹地区上年度职工月平均工资 × 6个月
```

**山西省数据查询**：
- 山西省统计局官网（http://tjj.shanxi.gov.cn/）
- 山西省人社厅官网
- 晋中市人力资源和社会保障局（如事故发生在晋中）

**示例计算**（基于山西省2024年数据）：
- 山西省2024年职工月平均工资：约7500元
- 丧葬补助金：7500 × 6 = 45,000元（约4.5万元）

### 供养亲属抚恤金计算
**计算公式**：
```
配偶抚恤金 = 死者本人工资 × 40%/月
其他亲属抚恤金（每人）= 死者本人工资 × 30%/月
孤寡老人或孤儿 = 上述标准 × 110%
抚恤金总额 = 各项抚恤金之和（不超过死者本人工资100%）
```

**领取条件**：
- 配偶：男满60周岁、女满55周岁
- 子女：未满18周岁或已满18周岁因残疾/上学不能独立生活
- 父母：男满60周岁、女满55周岁且无养老金
- 祖父母、外祖父母、孙子女：符合条件且无其他生活来源

**示例计算**：
假设死者生前月工资8000元，有配偶（55岁）和1名未成年子女（10岁）
- 配偶抚恤金：8000 × 40% = 3200元/月
- 子女抚恤金：8000 × 30% = 2400元/月
- 合计：5600元/月（未超过100%）

### 工亡赔偿总额计算
**计算公式**：
```
工亡赔偿总额 = 一次性工亡补助金 + 丧葬补助金 + 供养亲属抚恤金总额
```

**示例计算**（基于2024年数据）：
- 一次性工亡补助金：1,078,420元
- 丧葬补助金：45,000元
- 供养亲属抚恤金：5600元/月 × 12个月 × 10年（假设抚恤年限）= 672,000元
- 总计：约179.5万元

### 山西省年度标准更新查询
**查询渠道**：
1. 山西省统计局官网：查询上年度职工平均工资
2. 山西省人社厅官网：查询工伤保险待遇标准
3. 各市人社局官网：查询地方标准
4. 山西政务服务网：在线查询工伤保险待遇

**查询时间**：

- 建议每年更新计算标准

## 工亡案件诉讼策略与风险提示

### 仲裁前置程序
**时效规定**：
- 劳动仲裁：知道权利被侵害之日起1年
- 仲裁裁决：45日内（案情复杂可延长15日）
- 收到裁决书：15日内起诉

**诉讼衔接**：
- 收到仲裁裁决书15日内向法院起诉
- 追索工伤医疗费、供养亲属抚恤金可申请先予执行

### 证据收集注意事项
**时效性**：
- 微信聊天记录及时保存，防止对方撤回
- 银行流水及时打印，防止账户注销
- 监控录像及时调取，防止覆盖

**完整性**：
- 收集完整的工资发放记录（建议至少12个月）
- 收集完整的派单记录（建议最近3-6个月）
- 收集完整的工作群聊天记录

**合法性**"：
- 不得采用非法手段获取证据
- 录音录像需告知对方（特殊情况除外）
- 电子数据需固定取证时间

### 风险提示
**劳动关系认定风险**：
- 工资是微信个人转账，可能被抗辩为个人劳务
- 需证明发放工资人员是单位财务人员
- 需证明工资发放具有稳定性、持续性

**挂靠责任路径风险**：
- 需证明车辆以单位名义运营
- 需证明单位实际控制车辆
- 需排除其他可能的用工主体

**时效风险**：
- 工伤认定申请：1年内
- 劳动仲裁申请：1年内
- 交通事故诉讼：3年内
- 注意各环节时效衔接

**证据风险**：
- 部分证据可能在对方控制（如考勤记录）
- 及时申请法院调取对方掌握的证据
- 注意举证责任分配
