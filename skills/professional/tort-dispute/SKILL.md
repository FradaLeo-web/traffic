# 侵权纠纷律师技能

专注各类侵权纠纷，提供侵权认定、责任划分、损失计算等法律服务。

## 核心能力

- 侵权责任认定（过错责任、无过错责任、过错推定）
- 责任划分与承担
- 损害赔偿计算（人身、财产、精神）
- 侵权免责与减责
- 特殊侵权类型

## 1. 侵权责任构成

**一般侵权要件**：
- 加害行为（作为或不作为）
- 损害后果
- 因果关系
- 主观过错（故意或过失）

**归责原则**：
- 过错责任：一般侵权，由受害人证明过错
- 过错推定：特殊侵权，由加害人证明无过错
- 无过错责任：高度危险作业、产品责任、环境污染

## 2. 责任划分

**责任形态**：
- 单独责任：一人侵权
- 共同侵权：二人以上共同实施，承担连带责任
- 分别侵权：按份责任或连带责任
- 补充责任：安全保障义务人

**过失相抵**：
- 受害人对损害有过错，可减轻侵权人责任

## 3. 人身损害赔偿

**赔偿项目**：
| 项目 | 计算方式 |
|------|----------|
| 医疗费 | 挂号费、检查费、治疗费、手术费（凭发票） |
| 误工费 | 误工收入 × 误工期 |
| 护理费 | 护理标准 × 护理期 |
| 营养费 | 当地标准 × 营养期 |
| 交通费 | 就医交通费 |
| 残疾赔偿金 | 人均可支配收入 × 20年 × 伤残系数 |
| 精神损害抚慰金 | 5000-100000元（根据伤残等级） |

## 4. 精神损害赔偿

**适用情形**：
- 自然人人身权益（生命权、身体权、健康权等）
- 具有人格象征意义的特定纪念物品
- 侵权行为具有严重违法性

**赔偿因素**：
- 侵权人过错程度
- 侵害手段、场合、行为方式
- 造成的后果
- 侵权人获利情况
- 受诉法院所在地平均生活水平

## 5. 特殊侵权

**产品责任**：
- 生产者、销售者承担连带责任
- 消费者可向生产者或销售者主张

**医疗损害**：
- 医疗机构有过错、医疗行为与损害有因果关系
- 举证责任倒置

**环境污染**：
- 污染者行为与损害无因果关系的举证
