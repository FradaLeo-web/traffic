# 交通事故律师技能

专注道路交通事故侵权责任纠纷，处理从责任认定到赔偿执行的全流程法律事务。

## 核心能力

- 交通事故责任认定书分析与复核
- 人身损害赔偿精细化计算
- 交强险与商业险理赔
- 刑事责任评估（危险驾驶罪、交通肇事罪）
- 伤残等级预估

## 1. 责任分析

**事故形态**：
- 机动车vs机动车
- 机动车vs非机动车/行人
- 车辆自身事故

**责任划分**：全责、主责、同责、次责、无责

**法律依据**：《道路交通安全法》第76条

## 2. 赔偿项目

| 项目 | 计算方式 |
|------|----------|
| 医疗费 | 发票金额 + 后续治疗费 |
| 误工费 | 误工收入 × 误工期 |
| 护理费 | 护理标准 × 护理期 |
| 营养费 | 营养标准 × 营养期 |
| 交通费 | 实际凭证或酌定 |
| 住院伙食补助费 | 当地标准 × 住院天数 |
| 残疾赔偿金 | 人均可支配收入 × 20年 × 伤残系数 |
| 精神损害抚慰金 | 结合伤残等级酌定 |

## 3. 保险理赔

**交强险限额**（2024年新标准）：
- 死亡伤残：180,000元
- 医疗费用：18,000元
- 财产损失：2,000元

**理赔顺序**：先交强险 → 后商业险 → 最后侵权人

## 4. 刑事责任

**罪名定性**：
- 危险驾驶罪：醉驾（≥80mg/100ml）、追逐竞驶
- 交通肇事罪：致人死亡/重伤+主责以上

**量刑情节**：
- 加重：逃逸、酒驾、无证驾驶
- 减轻：自首、认罪认罚、取得谅解

## 5. 伤残等级

**分级标准**：《人体损伤致残程度分级》

**常见等级对应**：
- 一级：完全丧失劳动能力
- 五级：四肢大关节之一功能大部分丧失
- 十级：四肢长骨骨折后遗关节功能障碍

**鉴定时机**：治疗终结后3-6个月

## 6. 证据清单

- 事故责任认定书
- 病历、诊断证明、医疗费发票
- 司法鉴定意见书
- 收入证明（劳动合同、银行流水）
- 户口本、出生证明
