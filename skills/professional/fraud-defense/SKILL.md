# 诈骗罪辩护专业技能

专注诈骗类犯罪案件的辩护工作，包括普通诈骗、电信网络诈骗、合同诈骗等，提供从侦查到审判的全流程辩护支持。

## 核心能力

- **罪名辩护**：诈骗罪与相近罪名区分（盗窃、侵占、敲诈勒索）
- **定性辩护**：罪与非罪、此罪与彼罪
- **数额辩护**：犯罪数额认定、扣除成本
- **程序辩护**：管辖权异议、非法证据排除
- **量刑辩护**：从轻情节、缓刑适用
- **实务文书**：辩护词、取保申请、法律意见书

## 一、诈骗罪构成要件与辩护要点

### 1.1 诈骗罪法律规定

**《刑法》第266条**：
> 诈骗公私财物，数额较大的，处三年以下有期徒刑、拘役或者管制，并处或者单处罚金；数额巨大或者有其他严重情节的，处三年以上十年以下有期徒刑，并处罚金；数额特别巨大或者有其他特别严重情节的，处十年以上有期徒刑或者无期徒刑，并处罚金或者没收财产。

### 1.2 犯罪构成要件

| 要件 | 内容 | 辩护要点 |
|------|------|----------|
| 客体 | 公私财物所有权 | 是否存在真实财产权属争议 |
| 客观要件 | 欺诈行为→被害人错误认识→处分财产→损失 | 虚构事实/隐瞒真相是否达到"程度"要求 |
| 主体 | 一般主体（16岁以上） | 单位犯罪vs自然人犯罪 |
| 主观要件 | 直接故意，且以非法占有为目的 | 是否有非法占有目的、是否具有还款能力 |

### 1.3 数额认定标准（各地不同）

| 地区 | 数额较大 | 数额巨大 | 数额特别巨大 |
|------|----------|----------|--------------|
| 全国 | 3000-10000元 | 3万-10万 | 50万以上 |
| 北京 | 5000元 | 10万元 | 50万元 |
| 上海 | 5000元 | 10万元 | 50万元 |
| 山西 | 5000元 | 10万元 | 50万元 |
| 江苏 | 6000元 | 10万元 | 50万元 |
| 浙江 | 6000元 | 10万元 | 50万元 |

## 二、无罪辩护策略

### 2.1 民事纠纷vs诈骗罪

**关键区分**：是否存在"非法占有目的"

**辩护要点**：
1. **有真实的民事法律关系**
   - 存在真实的合同、债务关系
   - 有往来凭证、借条等证据
   
2. **有履约能力和意愿**
   - 具备还款能力
   - 有积极的还款行为
   - 虽未完全履约但非拒绝履约

3. **无欺骗行为或欺骗程度不够**
   - 未虚构身份、隐瞒真相
   - 虽有夸大但未达到"欺骗"程度

4. **未取得财物或取得合法**
   - 所获财物有合法依据
   - 属于民事争议范围

### 2.2 诈骗罪vs盗窃罪

**关键区分**：财物交付方式

| 情形 | 罪名 | 说明 |
|------|------|------|
| 秘密窃取 | 盗窃罪 | 被害人不知情 |
| 骗取后占有 | 诈骗罪 | 被害人"自愿"交付 |
| 欺骗与秘密结合 | 根据主要特征定罪 | 欺骗为主定诈骗 |

**辩护要点**：
- 被害人是否"自愿"处分财物
- 是否存在陷入错误认识
- 交付时是否具有处分意识

### 2.3 诈骗罪vs合同诈骗罪

**关键区分**：是否利用合同

| 罪名 | 欺骗方式 | 发生领域 |
|------|----------|----------|
| 诈骗罪 | 任何方式 | 任何领域 |
| 合同诈骗罪 | 利用经济合同 | 经济活动中 |

**辩护要点**：
- 是否签订真实合同
- 合同是否属于经济合同
- 是否在合同签订、履行过程中实施欺骗

## 三、电信网络诈骗辩护

### 3.1 电信诈骗特点

1. **集团化、组织化**：主犯、从犯区分
2. **层级分明**：不同层级量刑不同
3. **数额认定难**：被害人众多、跨境取证难
4. **从重情节多**：跨境、冒充公检法、未成年人

### 3.2 电信诈骗从重情节（《刑法》第266条之一）

| 从重情节 | 情节描述 |
|----------|----------|
| 拨打诈骗电话 | 拨打诈骗电话500人次以上 |
| 发送诈骗信息 | 发送诈骗信息5000条以上 |
| 诈骗手段恶劣 | 诈骗残疾人、老年人、未成年人、在校学生、低收入人群 |
| 造成严重后果 | 诈骗导致被害人死亡、精神失常等 |

### 3.3 电信诈骗辩护要点

1. **主从犯区分**
   - 组织者、领导者 → 主犯
   - 技术人员、底层话务员 → 从犯
   - 仅提供帮助行为 → 从犯

2. **犯罪数额认定**
   - 查实被害人的实际损失
   - 扣除未查实部分
   - 区分个人数额与参与数额

3. **立功情节**
   - 规劝同案犯投案
   - 提供线索抓获同案犯
   - 协助侦破其他案件

4. **认罪认罚**
   - 电信诈骗从宽幅度较大
   - 认罪认罚可减少基准刑60%以下

## 四、量刑辩护

### 4.1 法定/酌定从轻情节

| 情节类型 | 情节 | 量刑影响 |
|----------|------|----------|
| 法定从轻 | 自首 | 减少40%以下 |
| 法定从轻 | 坦白 | 减少20%以下 |
| 法定从轻 | 从犯 | 减少20%-50% |
| 法定从轻 | 未遂 | 减少50%以下 |
| 法定从轻 | 立功 | 减少20%以下 |
| 酌定从轻 | 退赃退赔 | 减少30%以下 |
| 酌定从轻 | 取得谅解 | 减少40%以下 |
| 酌定从轻 | 认罪认罚 | 减少30%以下 |
| 酌定从轻 | 初犯偶犯 | 从轻 |

### 4.2 缓刑适用

**适用条件**：
- 数额较大（3年以下）
- 犯罪情节较轻
- 有悔罪表现
- 无再犯罪危险
- 社区影响可控

**一般不适用缓刑**：
- 电信网络诈骗
- 诈骗老年人、残疾人、未成年人
- 造成严重后果

### 4.3 量刑计算示例

**案例**：普通诈骗罪，诈骗金额8万元

**情节**：
- 坦白
- 退赃5万元
- 认罪认罚

**计算**：
1. 量刑起点：2年（数额巨大起点）
2. 增加刑罚量：8万-5万=3万，增加6-8个月 → 基准刑2年6个月
3. 调节：
   - 坦白 -20%
   - 退赃 -20%
   - 认罪认罚 -20%
4. 最终：2.5年 × (1-20%-20%-20%) = 1.5年

**建议**：1-1.5年有期徒刑，可争取缓刑

## 五、程序辩护

### 5.1 管辖权异议

**情形**：
- 犯罪地与居住地不一致
- 电信诈骗由犯罪地公安机关管辖
- 被害人居住地法院管辖

### 5.2 非法证据排除

**可申请排除**：
- 刑讯逼供取得的供述
- 威胁、引诱、欺骗取得的供述
- 非法取得的物证、书证

### 5.3 取保候审申请

**适用条件**：
- 诈骗数额较小
- 患有严重疾病
- 怀孕或哺乳婴儿
- 认罪态度好、积极退赃

### 6.2 取保候审申请书要点

1. **申请理由**
   - 罪行较轻，可能判处管制、拘役或独立适用附加刑
   - 采取取保候审不致发生社会危险性

2. **保证措施**
   - 保证人保证（家属）
   - 保证金保证

3. **支持材料**
   - 病历（疾病）
   - 亲属关系证明（哺乳）
   - 退赃凭证

## 七、常见问题解答

### Q1：诈骗罪可以取保候审吗？
A：可以。符合《刑事诉讼法》第67条条件的，可以申请取保候审。诈骗罪属于可能判处管制、拘役或有期徒刑的案件，符合取保条件。

### Q2：诈骗罪退赃能减刑多少？
A：一般减少基准刑的30%以下。如果同时具有认罪认罚、取得谅解等情节，综合减少幅度可达60%。

### Q3：电信诈骗会判多重？
A：电信网络诈骗的量刑比普通诈骗更重。拨打诈骗电话500人次以上，发送诈骗信息5000条以上，属于"其他严重情节"，可处3-10年有期徒刑。

### Q4：诈骗罪可以不起诉吗？
A：可能。诈骗罪数额较大，且具有坦白、退赃、认罪认罚等情节，犯罪情节轻微的，可以争取相对不起诉。
