# 刑事辩护专业技能

专注刑事案件全流程辩护，从侦查阶段到审判执行，提供专业刑事法律支持。

## 核心能力

- **刑事会见策略**：律师首次会见谈话要点、权利义务告知
- **取保候审申请**：申请书撰写、保证措施设计
- **刑事证据分析**：证据三性审查、非法证据排除
- **量刑情节评估**：法定/酌定从轻减轻情节识别
- **庭审辩护策略**：辩护词、质证意见、发问提纲
- **刑事和解谈判**：被害人谅解、赔偿协议
- **上诉与申诉**：上诉状、申诉状、再审申请

## 一、刑事会见策略

### 1.1 会见准备

**证件与文书：**
- [ ] 律师事务所函
- [ ] 授权委托书（家属签署）
- [ ] 律师证复印件
- [ ] 会见笔录纸、印油

**法律文件：**
- [ ] 《刑事诉讼法》相关条款
- [ ] 《刑法》相关罪名条款
- [ ] 量刑指导意见相关条文

### 1.2 会见谈话要点

#### 第一部分：程序性事项
- **身份确认与委托关系建立**
- **诉讼权利告知**（拒绝回答无关问题、申请回避、聘请律师、申请变更强制措施等）
- **生活状况了解**（身体健康、药品需求、生活用品）

#### 第二部分：案情事实核实
- **涉案事实经过**（时间、地点、方式、金额）
- **主观故意认定**（动机、目的、是否打算归还）
- **退赃情况及原因**（款项去向、未退原因、退赔能力）
- **归案经过**（自首、抓获、如实供述）
- **侦查阶段供述情况**（讯问次数、笔录核对、有无刑讯逼供）

#### 第三部分：个人情况调查
| 调查项目 | 询问内容 | 法律意义 |
|----------|----------|----------|
| 基本情况 | 年龄、文化程度、工作经历、家庭情况 | 评估社会危险性 |
| 前科劣迹 | 是否受过刑事/行政处罚 | 判断是否构成累犯 |
| 工作表现 | 任职时间、职位、工作表现 | 评估主观恶性 |
| 退赃能力 | 财产状况、收入来源、债务情况 | 制定退赃方案 |
| 身体状况 | 是否患有严重疾病、是否特殊人群 | 取保候审依据 |

#### 第四部分：证据线索收集
- 对案件有利的证据或线索
- 侦查机关取证程序问题
- 证人联系方式

#### 第五部分：法律咨询与心理疏导
- 法律后果告知
- 取保候审可能性评估
- 心理安抚与诉讼指导

## 二、取保候审申请

### 2.1 申请条件分析

根据《刑事诉讼法》第67条，符合以下情形之一可申请取保候审：
1. 可能判处管制、拘役或者独立适用附加刑
2. 可能判处有期徒刑以上刑罚，采取取保候审不致发生社会危险性
3. 患有严重疾病、生活不能自理，怀孕或正在哺乳婴儿
4. 羁押期限届满，案件尚未办结

### 2.2 申请书核心要素

**申请理由重点：**
- 罪行较轻，可能判处拘役或三年以下有期徒刑
- 有悔罪表现，认罪态度良好
- 有固定住所，不会逃匿
- 积极退赃退赔，弥补被害人损失
- 无前科劣迹，初犯偶犯

**保证措施：**
- **保证人保证**：家属作为保证人，保证随传随到
- **保证金保证**：缴纳足额保证金

### 2.3 配套文书
- 取保候审申请书（模板位置：`knowledge-base/templates/取保候审申请书.md`）
- 保证人身份证明
- 退赃凭证（如有）

## 三、刑事证据分析

### 3.1 证据三性审查

| 审查维度 | 审查要点 | 常见问题 |
|----------|----------|----------|
| 真实性 | 证据来源是否合法、内容是否真实 | 伪造证据、变造证据 |
| 合法性 | 取证程序是否符合法律规定 | 刑讯逼供、诱供、违反程序取证 |
| 关联性 | 证据与案件事实的关联程度 | 无关证据、间接证据 |

### 3.2 非法证据排除

**可申请排除情形：**
1. 采用刑讯逼供等非法方法收集的犯罪嫌疑人、被告人供述
2. 采用暴力、威胁等非法方法收集的证人证言、被害人陈述
3. 违反法定程序收集的物证、书证，可能严重影响司法公正且不能补正或作出合理解释

**申请时机：**
- 侦查阶段：向侦查机关、检察机关提出
- 审查起诉阶段：向检察机关提出
- 审判阶段：庭前会议、法庭调查

## 四、量刑情节评估

### 4.1 法定从轻减轻情节

| 情节类型 | 法律规定 | 量刑影响 |
|----------|----------|----------|
| 自首 | 《刑法》第67条 | 可从轻或减轻处罚 |
| 立功 | 《刑法》第68条 | 可从轻或减轻处罚 |
| 从犯 | 《刑法》第27条 | 应当从轻、减轻或免除处罚 |
| 未遂 | 《刑法》第23条 | 可以比照既遂犯从轻或减轻处罚 |
| 坦白 | 《刑法》第67条第3款 | 可以从轻处罚 |

### 4.2 酌定从轻情节

1. **积极退赃退赔**：全额或部分退赃
2. **取得被害人谅解**：达成刑事和解
3. **初犯偶犯**：无前科劣迹
4. **认罪认罚**：签署认罪认罚具结书
5. **家庭困难**：需抚养老人、子女

### 4.3 量刑计算参考

根据《最高人民法院、最高人民检察院关于常见犯罪的量刑指导意见（试行）》：

**职务侵占罪**（17万元）：
- 量刑起点：6个月至1年有期徒刑
- 未退赃：增加基准刑的10%-30%
- 认罪认罚：减少基准刑的10%-30%
- 全额退赃并取得谅解：可争取缓刑

## 五、庭审辩护策略

### 5.1 辩护词结构

```
一、案件基本事实
   - 概括无争议事实
   - 明确争议焦点

二、法律适用分析
   - 罪名定性分析
   - 犯罪构成要件分析

三、证据质证意见
   - 证据三性分析
   - 证明力评估

四、量刑情节分析
   - 法定从轻减轻情节
   - 酌定从轻情节

五、辩护意见总结
   - 明确辩护诉求
   - 法律依据
```

### 5.2 发问提纲设计

| 问题类型 | 目的 | 示例 |
|----------|------|------|
| 诱导性问题 | 确认关键事实 | "当时你是否明确表示不同意？" |
| 开放性问题 | 获取详细信息 | "请描述一下当时的具体情况" |
| 时间线问题 | 重建事实顺序 | "在XX之前，你做了什么？" |
| 矛盾点问题 | 揭示陈述不一致 | "你刚才说A，但笔录中写的是B，请解释" |

### 5.3 质证技巧

1. **先实体后程序**：先审查证据内容，再审查取证程序
2. **重点突出**：针对关键证据深入质证
3. **法律依据**：引用相关法律、司法解释
4. **程序保障**：申请非法证据排除

## 六、刑事和解谈判

### 6.1 和解条件

1. **案件性质**：轻微刑事案件、有被害人案件
2. **犯罪嫌疑人**：认罪悔罪、愿意赔偿
3. **被害人**：愿意谅解、同意和解
4. **社会影响**：不会产生不良社会影响

### 6.2 谈判要点

**赔偿方案：**
- 赔偿金额：直接损失 + 精神抚慰金
- 支付方式：一次性支付、分期支付
- 履行期限：明确时间节点

**谅解书内容：**
- 对犯罪嫌疑人行为的谅解
- 同意司法机关从宽处理
- 赔偿款项已收到

## 七、上诉与申诉

### 7.1 上诉状撰写

**上诉理由类型：**
1. **事实认定错误**：关键事实未查清、证据不足
2. **法律适用错误**：罪名定性错误、量刑不当
3. **程序违法**：违反法定程序影响公正审判

### 7.2 申诉要点

**申诉条件：**
1. 有新证据证明原判决、裁定认定的事实确有错误
2. 据以定罪量刑的证据不确实、不充分或证明案件事实的主要证据之间存在矛盾
3. 原判决、裁定适用法律确有错误
4. 违反法定程序可能影响公正审判
