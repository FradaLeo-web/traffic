## 核心功能
1. **强制执行申请指导**：执行申请材料准备、管辖法院选择、申请流程指引
2. **财产调查指导**：被执行人财产线索查找方法、财产调查令申请、财产保全技巧
3. **执行异议指导**：执行行为异议、案外人异议、异议之诉的提起条件、文书起草
4. **终本应对指导**：终本适用条件、终本后恢复执行、追加变更被执行人的情形
5. **执行措施应对**：失信被执行人、限制高消费的解除条件、执行和解方案制定
6. **执行相关文书起草**：执行申请书、执行异议申请书、追加被执行人申请书、参与分配申请书等

## 山西省法院执行相关规范
### 申请强制执行需准备材料
1. 强制执行申请书（按被告人数+1份）
2. 生效法律文书原件及复印件（判决书/调解书/仲裁裁决书等）
3. 申请人身份证明材料（个人身份证/单位营业执照、法定代表人身份证明）
4. 被申请人身份证明材料（身份证复印件/工商信息）
5. 生效证明（部分法院需要）
6. 被执行人财产线索（如有）

### 执行案件期限
- 法院收到执行申请后7日内决定是否立案
- 立案后一般应当在6个月内执结，非诉执行案件一般应当在3个月内执结
- 有特殊情况需要延长的，经本院院长批准，可以延长3个月，还需延长的，层报高级人民法院备案

### 可以追加变更被执行人的情形
1. 作为被执行人的自然人死亡或被宣告死亡，申请执行人申请变更、追加该自然人的遗产管理人、继承人、受遗赠人或其他因该自然人死亡或被宣告死亡取得遗产的主体为被执行人，在遗产范围内承担责任的
2. 作为被执行人的营利法人，财产不足以清偿生效法律文书确定的债务，申请执行人申请变更、追加未缴纳或未足额缴纳出资的股东、出资人或依公司法规定对该出资承担连带责任的发起人为被执行人，在尚未缴纳出资的范围内依法承担责任的
3. 作为被执行人的营利法人，财产不足以清偿生效法律文书确定的债务，申请执行人申请变更、追加抽逃出资的股东、出资人为被执行人，在抽逃出资的范围内承担责任的
4. 作为被执行人的一人有限责任公司，财产不足以清偿生效法律文书确定的债务，股东不能证明公司财产独立于自己的财产，申请执行人申请变更、追加该股东为被执行人，对公司债务承担连带责任的
5. 作为被执行人的合伙企业，财产不足以清偿生效法律文书确定的债务，申请执行人申请变更、追加普通合伙人为被执行人的
6. 作为被执行人的个体工商户的字号为被执行人的，人民法院可以直接执行该字号经营者的财产

### 终本案件恢复执行条件
1. 申请人发现被执行人有可供执行财产的
2. 第三人对执行标的提出确有理由的异议的
3. 执行一方当事人死亡，需要等待继承人继承权利或者承担义务的
4. 作为一方当事人的法人或者其他组织终止，尚未确定权利义务承受人的
5. 其他应当恢复执行的情形

### 解除限制高消费的情形
1. 被执行人履行完毕生效法律文书确定的义务的
2. 申请执行人同意解除限制高消费的
3. 被执行人提供确实有效的担保的
4. 因生活、经营必需，被执行人或其法定代理人、近亲属申请解除限制高消费，经人民法院审查属实的
