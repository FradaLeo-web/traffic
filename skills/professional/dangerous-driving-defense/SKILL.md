# 危险驾驶罪辩护专业技能

专注危险驾驶类犯罪案件的辩护工作，包括醉酒驾驶、追逐竞驶、严重超速等，提供从侦查到审判的全流程辩护支持。

## 核心能力

- **无罪辩护**：血检程序违法、含量计算错误
- **程序辩护**：血样提取、鉴定程序
- **量刑辩护**：从轻情节、缓刑适用
- **情节辩护**：从重情节认定、反驳
- **不起诉辩护**：情节轻微、认罪认罚
- **实务文书**：辩护词、法律意见书

---

## 一、危险驾驶罪构成要件

### 1.1 法律规定

**《刑法》第133条之一**：
> 在道路上驾驶机动车，有下列情形之一的，处拘役，并处罚金：
> （一）追逐竞驶，情节恶劣的；
> （二）醉酒驾驶机动车的；
> （三）从事校车业务或者旅客运输，严重超过额定乘员载客，或者严重超过规定时速行驶的；
> （四）违反危险化学品安全管理规定运输危险化学品，危及公共安全的。

### 1.2 追诉标准

| 情形 | 立案标准 |
|------|----------|
| 追逐竞驶 | 情节恶劣（时速超过50%或3人以上追逐） |
| 醉酒驾驶 | 血液酒精含量≥80mg/100ml |
| 严重超员 | 超员100%以上或额定乘员10人以上 |
| 严重超速 | 超速50%以上 |
| 危险化学品 | 危及公共安全 |

## 二、醉驾辩护策略

### 2.1 无罪/不作犯罪处理

**情节显著轻微，不作为犯罪处理**（血液酒精含量<120mg/100ml）：
- 未造成实际损害后果
- 认罪、悔罪
- 不具有从重处罚情形

**犯罪情节轻微，可免予刑事处罚**：
- 血液酒精含量<120mg/100ml
- 非高速公路
- 无事故或事故轻微
- 认罪认罚

### 2.2 血检程序辩护（重点）

**血样提取程序违法**：
1. **提取主体**
   - 应由医护人员或法医提取
   - 警察提取 → 程序违法
   
2. **消毒用品**
   - 应使用醇类药品消毒
   - 使用碘酒、酒精消毒 → 可能影响结果
   
3. **储存条件**
   - 低温冷藏（2-8℃）
   - 使用抗凝管
   - 违规储存 → 可能导致结果偏差

**血检鉴定程序违法**：
1. **鉴定资质**
   - 鉴定机构应具备资质
   - 鉴定人应具备资格
   
2. **鉴定方法**
   - 应使用GA/T1073或GA/T842方法
   - 超过6个月重新检测
   
3. **送检流程**
   - 不得由当事人自行送检
   - 应由办案机关送检

### 2.3 含量计算辩护

**辩护要点**：
1. **时间差**
   - 驾驶时间与抽血时间的间隔
   - 间隔越长，驾驶时含量可能越低
   
2. **代谢因素**
   - 个体差异
   - 体重、性别、年龄
   - 是否空腹
   
3. **证据矛盾**
   - 当事人陈述与血检结果矛盾
   - 其他证据印证当事人陈述

## 三、从重情节认定与反驳

### 3.1 法定从重情节（《意见》第2条）

| 从重情节 | 辩护要点 |
|----------|----------|
| 造成交通事故且负全部/主要责任 | 事故责任认定书是否为全部/主要责任 |
| 造成交通事故后逃逸 | 是否有逃逸情节 |
| 血液酒精含量≥200mg/100ml | 是否有复核的必要 |
| 高速公路驾驶 | 是否有证据证明 |
| 组织/参与追逐竞驶 | 是否有组织行为 |
| 吸食、注射毒品 | 是否有相关证据 |
| 驾驶营运机动车 | 是否为营运车辆 |
| 严重超员/超载/超速 | 是否有鉴定意见 |
| 无驾驶资格 | 是否有证据证明 |
| 使用伪造/变造车牌 | 是否有证据 |
| 逃避检查 | 是否有拒绝检查行为 |
| 曾因酒驾受处罚 | 行政处罚决定书是否生效 |

### 3.2 从重情节反驳策略

| 从重情节 | 反驳证据/理由 |
|----------|---------------|
| 事故责任 | 复核事故认定书责任划分 |
| 高速驾驶 | 提供行车路线证据 |
| 无证驾驶 | 是否有临时驾驶能力 |
| 逃避检查 | 是否存在合理怀疑 |
| 曾受处罚 | 处罚是否已生效、是否已过追诉时效 |

## 四、量刑辩护

### 4.1 量刑起点

**全国标准**：1-6个月拘役

**山西省标准**：
| 情形 | 量刑起点 |
|------|----------|
| 血液酒精<120mg/100ml | 1个月 |
| 血液酒精120-200mg/100ml | 2个月 |
| 血液酒精≥200mg/100ml | 3个月 |
| 有从重情节 | 增加1-2个月 |

### 4.2 从轻情节

| 情节 | 减少幅度 |
|------|----------|
| 认罪认罚 | 30%以下 |
| 坦白 | 20%以下 |
| 血液酒精含量<120mg/100ml | 可以不作为犯罪处理 |
| 积极赔偿 | 20%以下 |
| 取得被害人谅解 | 20%以下 |

### 4.3 缓刑适用

**适用条件**：
- 血液酒精含量<160mg/100ml
- 无从重情节
- 认罪悔罪
- 符合社区矫正条件

**一般不适用缓刑**（有从重情节）：
- 酒精含量≥200mg/100ml
- 高速公路驾驶
- 造成交通事故且逃逸
- 驾驶营运机动车
- 曾因酒驾受刑事/行政处罚

### 4.4 罚金计算

| 拘役期限 | 罚金金额 |
|----------|----------|
| 1个月 | 2000-5000元 |
| 2个月 | 4000-8000元 |
| 3个月 | 6000-15000元 |
| 4个月 | 8000-20000元 |
| 5-6个月 | 10000-30000元 |

## 五、不起诉辩护

### 5.1 不起诉条件

**相对不起诉**（情节轻微）：
- 血液酒精含量<120mg/100ml
- 未造成损害后果
- 认罪认罚
- 无从重情节

**证据不足不起诉**：
- 血检程序违法
- 鉴定意见存疑
- 证据链不完整

### 5.2 不起诉申请材料

1. 法律意见书
2. 相关证据材料
3. 认罪认罚具结书（如有）
4. 被害人谅解（如有）

## 六、实务要点

### 6.1 辩护词要点

```
一、案件基本情况
二、辩护意见
   （一）关于事实认定
   （二）关于从重情节的认定
   （三）关于量刑
三、量刑情节
   - 认罪认罚
   - 血液酒精含量较低
   - 无前科
四、建议
   - 不起诉/缓刑/从轻处罚
```

### 6.2 取保候审申请

**适用条件**：
- 血液酒精含量<150mg/100ml
- 无从重情节
- 认罪态度好
- 符合社区矫正条件

## 七、常见问题解答

### Q1：醉驾取保候审容易吗？
A：相对容易。血液酒精含量<150mg/100ml、无事故、无从重情节的，一般可以取保候审。

### Q2：血检结果有异议怎么办？
A：可以申请重新鉴定或对鉴定意见提出异议。重点审查：血样提取程序、储存条件、鉴定资质、鉴定方法。

### Q3：醉驾可以不起诉吗？
A：可以。血液酒精含量<120mg/100ml、未造成损害后果、认罪认罚、无从重情节的，可以争取相对不起诉。

### Q4：醉驾缓刑容易吗？
A：酒精含量<160mg/100ml、无从重情节、认罪认罪，可以争取缓刑。酒精含量>200mg/100ml的，一般不适用缓刑。

### Q5：吹气检测和血检哪个为准？
A：血检为准。吹气检测仅作为初筛，最终以血检鉴定结果为准。
