# 合同纠纷律师技能

专注各类合同纠纷，提供合同审查、违约认定、损失计算等法律服务。

## 核心能力

- 合同效力审查（有效、无效、可撤销、效力待定）
- 违约责任认定
- 损失赔偿计算
- 合同解除与终止
- 合同条款解释

## 1. 合同效力分析

**有效要件**：
- 当事人具有相应民事行为能力
- 意思表示真实
- 不违反法律、行政法规的强制性规定
- 不违背公序良俗

**无效情形**：
- 恶意串通，损害他人合法权益
- 以合法形式掩盖非法目的
- 违反法律、行政法规的强制性规定
- 损害社会公共利益

**可撤销**：
- 重大误解
- 显失公平
- 欺诈、胁迫
- 乘人之危

**法律依据**：《民法典》第143条、第153-154条、第147-151条

## 2. 违约责任

**违约类型**：
- 迟延履行（逾期付款、逾期交付）
- 不适当履行（质量不符、数量不符）
- 不完全履行（部分履行）
- 拒绝履行

**归责原则**：
- 严格责任：违约即承担
- 过错责任：仅在有过错时承担

**免责事由**：
- 不可抗力
- 对方过错
- 合同约定免责条款

## 3. 损失赔偿

**赔偿范围**：
- 实际损失：因违约所遭受的财产减损
- 可得利益：合同履行后可获得的利益

**计算示例**：
- 买卖合同：货款+利息+替代交易差价
- 租赁合同：租金+违约金+恢复原状费用
- 借款合同：本金+利息+违约金（总计不超过LPR4倍）

**违约金调整**：
- 约定过高（超过损失30%），可请求减少
- 约定过低（低于损失30%），可请求增加

## 4. 合同解除

**解除方式**：
- 法定解除：不可抗力、预期违约、迟延履行、根本违约
- 约定解除：合同约定的解除条件成就

**解除后果**：
- 尚未履行：终止履行
- 已履行：根据履行情况和合同性质，请求恢复原状、采取补救措施、赔偿损失

## 5. 证据清单

- 书面合同、补充协议
- 付款凭证（转账记录、收据）
- 交付凭证（物流单、签收单）
- 质量检验报告、验收单
- 沟通记录（邮件、聊天记录）
