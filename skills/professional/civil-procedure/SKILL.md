# 民事诉讼通用技能

专注民事诉讼全流程的程序性问题。

## 核心能力

- 管辖法院选择
- 立案材料准备
- 举证责任分配
- 庭审程序应对
- 判决申请执行

## 1. 管辖法院

**级别管辖**：
- 基层法院：一般民事案件
- 中级法院：重大影响案件、涉外案件

**地域管辖**：
- 一般原则：被告住所地法院
- 合同纠纷：被告住所地或合同履行地
- 侵权纠纷：侵权行为地或被告住所地
- 不动产纠纷：不动产所在地（专属管辖）

**协议管辖**：
- 合同当事人可书面协议选择管辖法院
- 限于被告住所地、合同履行地、合同签订地、原告住所地、标的物所在地

## 2. 立案材料

**起诉状要求**：
- 原告信息：姓名、性别、住址、联系方式
- 被告信息：姓名/名称、住址、联系方式
- 诉讼请求：明确、具体
- 事实与理由：分段叙述，引用法条
- 证据和证据来源

**证据材料**：
- 证据目录（名称、来源、证明目的）
- 证据复印件
- 原始证据（开庭时携带）

**身份证明**：
- 自然人：身份证复印件
- 法人：营业执照、法定代表人身份证明
- 代理人：授权委托书

## 3. 举证责任

**原则**：谁主张谁举证

**倒置情形**：
- 医疗损害：医疗机构有过错
- 高空坠物：所有人、管理人无过错
- 饲养动物：受害人故意或重大过失
- 产品缺陷：生产者无缺陷

**期限**：法院指定不少于15日

## 4. 庭审程序

**流程**：
1. 开庭准备：核对身份、告知权利义务
2. 法庭调查：原告陈述→被告答辩→举证质证
3. 法庭辩论：原告发言→被告发言→互相辩论
4. 最后陈述

**证人询问**：
- 封闭式问题（是/否回答）
- 从背景信息到关键事实
- 预测对方攻击点并准备应对

## 5. 执行申请

**申请条件**：
- 生效判决书
- 履行期限届满
- 申请时效：2年

**执行措施**：
- 查询、冻结、划拨被执行人存款
- 查封、扣押、拍卖被执行人财产
- 限制高消费、纳入失信名单

**财产保全**：
- 诉前保全：紧急情况下
- 诉中保全：诉讼过程中
