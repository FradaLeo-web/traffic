# 量刑计算器

基于最高人民法院、最高人民检察院《关于常见犯罪的量刑指导意见（试行）》及各地实施细则，自动计算量刑结果。

## 核心功能

- 量刑起点计算：根据基本犯罪构成事实确定量刑起点
- 基准刑计算：在量刑起点基础上增加刑罚量
- 宣告刑计算：根据量刑情节调节基准刑
- 缓刑预测：评估缓刑适用可能性
- 罚金计算：根据犯罪情节确定罚金数额

---

## 一、量刑计算步骤

### 步骤1：确定基本犯罪构成事实

请提供以下信息：
- 涉嫌罪名
- 案件基本事实（犯罪数额、犯罪次数、危害后果）
- 责任认定（主责/全部责任/同等责任/次要责任）

### 步骤2：确定量刑起点

根据《量刑指导意见》，在相应法定刑幅度内确定量刑起点：

| 罪名 | 情节 | 量刑起点 |
|------|------|----------|
| 危险驾驶罪 | 醉酒驾驶 | 1个月拘役 |
| 盗窃罪 | 数额较大（1000-3000元起） | 1年以下有期徒刑 |
| 诈骗罪 | 数额较大（3000-10000元起） | 1年以下有期徒刑 |
| 故意伤害罪 | 致人轻伤 | 2年以下有期徒刑 |
| 交通肇事罪 | 死亡一人或重伤三人 | 1-2年有期徒刑 |

### 步骤3：计算基准刑

在量刑起点基础上，根据其他犯罪事实增加刑罚量：

```
基准刑 = 量刑起点 + 增加刑罚量
```

### 步骤4：量刑情节调节

#### 从重处罚情节

| 情节 | 调节比例 |
|------|----------|
| 累犯 | 增加基准刑10%-40% |
| 前科 | 增加基准刑10%以下 |
| 犯罪对象为未成年人/老年人/残疾人 | 增加基准刑20%以下 |
| 避重就轻、推卸责任 | 从严掌握 |

#### 从轻处罚情节

| 情节 | 调节比例 |
|------|----------|
| 自首 | 减少基准刑40%以下 |
| 坦白 | 减少基准刑20%以下 |
| 立功 | 减少基准刑20%以下 |
| 认罪认罚 | 减少基准刑30%以下 |
| 从犯 | 减少基准刑20%-50% |
| 未遂 | 减少基准刑50%以下 |
| 退赃退赔 | 减少基准刑30%以下 |
| 取得被害人谅解 | 减少基准刑40%以下 |

### 步骤5：确定宣告刑

```
宣告刑 = 基准刑 × (1 - 从轻情节调节比例 + 从重情节调节比例)
```

调节结果在20%幅度内调整。

## 二、常见罪名量刑参数（全国标准）

### 1. 危险驾驶罪

**量刑起点**：1-6个月拘役

**从重情节**（血液酒精含量）：
| 情形 | 从重幅度 |
|------|----------|
| 80-120mg/100ml | 基准 |
| 120-200mg/100ml | 从重 |
| 200mg/100ml以上 | 严重从重 |

**从重情节**（其他）：
- 造成交通事故且负事故全部或主要责任
- 高速公路驾驶
- 驾驶营运机动车
- 曾因酒驾受过行政处罚或刑事追究

**罚金**：
- 拘役3个月以下：2000-10000元
- 拘役3-6个月：5000-20000元

**缓刑适用**：
- 酒精含量<120mg/100ml，无从重情节 → 可适用缓刑
- 有从重情节 → 一般不适用缓刑

### 2. 盗窃罪

**量刑起点**（数额较大）：
| 地区 | 起点金额 |
|------|----------|
| 全国 | 1000-3000元 |
| 北京 | 1000元 |
| 上海 | 1000元 |
| 深圳 | 1000元 |
| 山西 | 2000元 |

**量刑幅度**：
| 数额档次 | 量刑起点 | 最高刑 |
|----------|----------|--------|
| 数额较大（1千-3千） | 1年以下 | 3年 |
| 数额巨大（3万-10万） | 3年 | 10年 |
| 数额特别巨大（30万-50万） | 10年 | 无期 |

**调节规则**：
- 每增加2000-3000元，增加1个月刑期
- 扒窃、入室盗窃、多次盗窃 → 从重

### 3. 诈骗罪

**量刑起点**（数额较大）：
| 地区 | 起点金额 |
|------|----------|
| 全国 | 3000-10000元 |
| 北京 | 5000元 |
| 上海 | 5000元 |
| 山西 | 5000元 |

**量刑幅度**：
| 数额档次 | 量刑起点 | 最高刑 |
|----------|----------|--------|
| 数额较大（5千-5万） | 3年以下 | 3年 |
| 数额巨大（5万-50万） | 3-10年 | 10年 |
| 数额特别巨大（50万以上） | 10年 | 无期 |

**调节规则**：
- 每增加5000-10000元，增加1个月刑期
- 电信网络诈骗 → 从重20%-40%

### 4. 故意伤害罪

**量刑起点**：
| 伤情 | 量刑起点 |
|------|----------|
| 轻伤一级 | 1-2年 |
| 轻伤二级 | 6个月-1年 |
| 重伤一级 | 3-5年 |
| 重伤二级 | 2-4年 |

**调节规则**：
- 每增加轻伤一级，增加6个月
- 每增加重伤一级，增加1-2年

**从重情节**：
- 持刀、持枪等凶器
- 故意伤害未成年人、老年人
- 致人多处伤残

### 5. 交通肇事罪

**量刑起点**：
| 事故责任 | 后果 | 量刑起点 |
|----------|------|----------|
| 主要责任 | 死亡1人 | 1-1.6年 |
| 全部责任 | 死亡1人 | 1.6-2年 |
| 同等责任 | 死亡3人 | 1.6-2年 |

**逃逸情节**：
- 肇事后逃逸：3-5年
- 逃逸致人死亡：7-9年

### 6. 非法吸收公众存款罪

**量刑起点**：
| 数额 | 对象 | 量刑起点 |
|------|------|----------|
| 100万元 | 150人 | 1年以下 |
| 500万元 | 500人 | 3-4年 |
| 5000万元 | 5000人 | 10-11年 |

## 三、量刑情节计算示例

### 示例1：危险驾驶罪（醉驾）

**案件信息**：
- 罪名：危险驾驶罪
- 血液酒精含量：160mg/100ml
- 行驶路段：城市道路（非高速）
- 后果：发生交通事故，负主要责任
- 既往：曾因酒驾被行政处罚
- 态度：认罪认罚

**计算过程**：

1. **量刑起点**：2个月拘役

2. **从重情节**：
   - 酒精含量160mg/100ml → 从重
   - 负主要责任 → 从重
   - 曾受行政处罚 → 从重
   - 累计增加：3-4个月

3. **从轻情节**：
   - 认罪认罚 → 减少20%

4. **计算**：
   - 基准刑：2 + 3.5 = 5.5个月
   - 调节后：5.5 × (1-20%) = 4.4个月

**结果**：
- 宣告刑：4个月拘役
- 罚金：8000元
- 缓刑：适用难度大（有从重情节）

---

### 示例2：盗窃罪

**案件信息**：
- 罪名：盗窃罪
- 盗窃金额：15000元（山西标准，数额巨大起点3万，数额较大2千）
- 情节：多次盗窃、入室盗窃
- 态度：坦白、已退赃
- 前科：无

**计算过程**：

1. **量刑起点**：1年（数额较大）

2. **增加刑罚量**：
   - 数额：15000元，增加4-6个月
   - 多次盗窃、入室盗窃 → 从重

3. **量刑情节**：
   - 坦白 → 减少20%
   - 退赃 → 减少20%

4. **计算**：
   - 基准刑：1年 + 5个月 = 17个月
   - 调节后：17 × (1-20%-20%) = 10.2个月

**结果**：
- 宣告刑：10个月有期徒刑
- 缓刑：可以考虑

## 四、缓刑适用评估

### 缓刑适用条件（《刑法》第72条）

1. 判处三年以下有期徒刑或者拘役
2. 犯罪情节较轻
3. 有悔罪表现
4. 没有再犯罪的危险
5. 宣告缓刑对所居住社区没有重大不良影响

### 应当适用缓刑

- 不满18周岁的人
- 怀孕的妇女
- 已满75周岁的人

### 一般不适用缓刑

| 罪名 | 不适用情形 |
|------|------------|
| 危险驾驶 | 酒精≥200mg/ml、高速公路、事故逃逸 |
| 盗窃 | 扒窃、惯窃、入室盗窃情节严重 |
| 诈骗 | 电信网络诈骗、造成严重后果 |
| 交通肇事 | 逃逸致人死亡 |
