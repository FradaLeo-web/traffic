# 民事诉讼通用技能

专注于民事诉讼全流程的程序性问题，包括立案、管辖、证据、庭审、执行等程序法问题。

## 核心能力

- 管辖法院选择（级别管辖、地域管辖、协议管辖、专属管辖）
- 立案材料准备（起诉状、证据清单、身份证明）
- 举证责任分配（谁主张谁举证、举证责任倒置）
- 庭审程序应对（举证质证、法庭辩论、最后陈述）
- 判决申请执行（申请执行、财产保全、执行措施）

## 输出格式规范

### 1. 管辖法院确定

**级别管辖**：
- 基层法院：一审普通民事案件
- 中级法院：重大影响案件、涉外案件、本辖区有重大影响案件
- 高级法院：本辖区有重大影响案件
- 最高法院：全国有重大影响案件

**地域管辖**：
- 一般原则：被告住所地法院
- 特殊规定：
  - 合同纠纷：被告住所地或合同履行地
  - 侵权纠纷：侵权行为地或被告住所地
  - 不动产纠纷：不动产所在地法院（专属管辖）
  - 公司诉讼：公司住所地法院
  - 保险合同纠纷：被告住所地或保险标的物所在地

**协议管辖**：
- 合同或其他财产权益纠纷当事人可以书面协议选择
- 限于被告住所地、合同履行地、合同签订地、原告住所地、标的物所在地
- 不得违反级别管辖、专属管辖

### 2. 立案材料清单

**起诉状要求**：
- 原告信息：姓名、性别、民族、出生日期、住址、联系方式
- 被告信息：姓名/名称、住址/住所地、联系方式
- 诉讼请求：明确、具体（如请求判决被告赔偿原告经济损失××元）
- 事实与理由：分段叙述事实经过，引用法律条文
- 证据和证据来源，证人姓名和住址
- 此致××法院、具状人、日期

**证据材料**：
- 证据目录（证据名称、证据来源、证明目的）
- 证据复印件（按顺序整理）
- 原始证据（开庭时携带）

**身份证明**：
- 自然人：身份证复印件
- 法人/其他组织：营业执照、法定代表人身份证明
- 授权委托书（如委托代理人）

**其他材料**：
- 管辖权证明（被告住所地证明、合同履行地证明等）
- 先予执行申请书（如需）
- 财产保全申请书（如需）

### 3. 举证责任分配

**一般原则：谁主张谁举证**
- 主张法律关系存在：承担举证责任
- 主张法律关系变更、消灭、权利受到妨害：承担举证责任

**举证责任倒置**（特殊情形）：
- 医疗损害：医疗机构有过错、医疗行为与损害有因果关系
- 环境污染：污染者行为与损害无因果关系
- 高空坠物：所有人、管理人使用人无过错
- 饲养动物：受害人故意或重大过失
- 产品缺陷：生产者无缺陷

**举证期限**：
- 法院指定举证期限（不得少于15日）
- 逾期提供证据：法院可能不予采纳（存在正当理由除外）

### 4. 庭审程序流程

**开庭准备**：
- 法官/书记员核对当事人身份
- 宣布案由、合议庭组成人员、书记员
- 告知当事人诉讼权利义务
- 询问是否申请回避

#### 证人询问提纲生成（【新增功能】）

根据证人陈述，生成结构化询问提纲：
- **问题顺序建议**：从背景信息到关键事实，逻辑递进
- **每个问题预计要引出的证言要点**：明确每个问题的目的
- **对方在交叉询问时可能提出的攻击点**：预测对方可能攻击的问题
- **针对这些攻击点的应对建议**：提供应对交叉询问的策略

输出要求：以法院庭审风格输出，问题设计为封闭式问题（是/否回答）

#### 预测对方诉讼策略（【新增功能】）

基于案情，为被告方推演可能诉讼策略：
- **核心论点**：每套策略的核心观点
- **需要依赖的关键事实或证据**：支撑策略的事实基础
- **可能引用的法律条文或案例**：策略的法律依据
- **原告方对此策略的弱点与应对方式**：分析对方策略的弱点及我方应对

输出要求：结构化呈现，推演3-5套可能策略

**法庭调查**：
1. 原告陈述诉讼请求、事实与理由
2. 被告答辩
3. 原告举证
4. 被告质证
5. 被告举证
6. 原告质证
7. 法官询问（必要时）
8. 法院调取证据（如有）

**质证要点**：
- 证据真实性（是否真实、是否伪造）
- 证据合法性（取得方式是否合法、形式是否合法）
- 证据关联性（与待证事实是否有联系）
- 证明力大小

**法庭辩论**：
1. 原告发言
2. 被告发言
3. 互相辩论
4. 辩论顺序

**最后陈述**：
- 原告最后陈述
- 被告最后陈述

**法庭调解**：
- 庭审后询问是否愿意调解
- 调解不成，择日宣判

### 5. 判决类型与生效

**判决类型**：
- 判决书：对实体问题的裁决（如确认权利、判决履行义务）
- 裁定书：对程序问题的裁决（如驳回起诉、管辖权异议）
- 调解书：双方达成调解协议

**判决生效时间**：
- 一审判决：收到判决书之日起15日内未上诉
- 二审判决：判决书送达之日生效
- 调解书：签收之日生效

**上诉期限**：
- 一审判决：15日
- 一审裁定：10日

### 6. 申请执行

**申请执行条件**：
- 生效法律文书（判决书、调解书、仲裁裁决）
- 义务人未履行义务
- 申请执行期限：2年（自法律文书规定履行期间的最后一日起计算）

**申请执行材料**：
- 申请执行书
- 生效法律文书（原件）
- 申请执行人身份证明
- 被执行人财产线索（如有）

**执行措施**：
- 查封、冻结、扣押财产
- 拍卖、变卖财产
- 划拨存款
- 强制迁出房屋、退出土地
- 限制高消费
- 纳入失信被执行人名单
- 拘留、罚款

**执行和解**：
- 双方达成执行和解协议
- 暂缓执行、分期履行、变更履行方式
- 和解不成，恢复执行

### 7. 财产保全

**诉前保全**：
- 条件：情况紧急，不立即保全将使权益受到难以弥补的损害
- 担保：应当提供担保
- 时效：保全后30日内起诉，否则解除保全

**诉中保全**：
- 条件：可能因当事人一方的行为或其他原因，使判决难以执行
- 担保：可以责令提供担保
- 范围：限于请求的范围，或与本案有关的财物

**保全方式**：
- 查封（不动产）
- 扣押（动产）
- 冻结（银行存款、债权）

**保全解除**：
- 被申请人提供担保
- 申请人撤回保全申请
- 保全错误（申请人赔偿损失）

### 8. 诉讼风险提示

**程序风险**：
- 超过诉讼时效（3年）
- 超过上诉期限（15日/10日）
- 超过申请执行期限（2年）
- 违反管辖规定（驳回起诉或移送管辖）

**实体风险**：
- 举证不能（承担不利后果）
- 事实认定错误
- 法律适用不当

**执行风险**：
- 被执行人无财产可供执行
- 财产权属争议
- 执行异议

**应对措施**：
- 及时起诉，避免超过时效
- 全面收集证据，形成证据链
- 诉前/诉中财产保全
- 持续追踪被执行人财产线索
