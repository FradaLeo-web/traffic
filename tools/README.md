# 工具脚本目录

本目录用于存放辅助工具脚本，包括：

## 可用工具

- **verify-knowledge-base.sh**：知识库完整性验证工具
  - 检查目录结构
  - 统计文件数量
  - 验证关键文件存在性
  - 生成验证报告

- **知识库管理工具**（待开发）：批量更新知识库、验证格式
- **文件转换工具**（待开发）：Word转Markdown、格式规范化
- **自动化脚本**（待开发）：定期更新计算标准、同步法律数据

## 使用说明

### 知识库验证工具

```bash
# 运行验证脚本
bash tools/verify-knowledge-base.sh

# 或（在Linux/Mac上）
chmod +x tools/verify-knowledge-base.sh
./tools/verify-knowledge-base.sh
```

**验证内容**：
1. 目录结构完整性
2. 文件数量统计
3. 关键文件存在性检查
4. 案例分类验证
5. 标准文件检查
6. 模板文件检查

**输出说明**：
- ✓ 绿色：检查通过
- ✗ 红色：检查失败
- ⚠ 黄色：警告信息

工具脚本采用Python或Shell编写，具体使用方法请参考各脚本的注释说明。
